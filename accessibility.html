Accessibility Statement

Last reviewed: 1 January 2025

EU4Reconstruction is committed to ensuring that this website is accessible to everyone, including people with disabilities. We strive to meet the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA.

Our Commitment

We believe that access to information about reconstruction funding and programme activities is a right for all people. The design of this website follows inclusive principles:

  • All images have descriptive alternative text
  • All form inputs have associated labels
  • Colour contrast ratios meet or exceed WCAG 2.1 AA thresholds (minimum 4.5:1 for normal text, 3:1 for large text)
  • All interactive elements are keyboard accessible and have visible focus indicators
  • The website is navigable using a screen reader with appropriate ARIA roles and labels
  • A skip-to-content link is provided at the top of every page
  • The mobile navigation menu includes focus trapping and Escape key closure
  • No content flashes more than three times per second
  • Text can be resized up to 200% without loss of content or functionality
  • Animations respect the prefers-reduced-motion media query

Conformance Status

This website aims to be partially conformant with WCAG 2.1 Level AA. Partial conformance means that some parts of the content do not fully conform to the accessibility standard.

Known limitations at this time:

  • Some PDF documents linked from this website (grant guidelines, templates) may not be fully accessible. We are working to remediate these documents. Please contact us if you require an accessible version of any document.
  • Partner logos displayed on the Partners page are currently placeholder images without meaningful alt text in the production design. These will be updated when real logos are provided.

Technical Specifications

This website relies on the following technologies for its accessibility:

  • HTML5 semantic markup
  • CSS (without reliance on colour alone to convey meaning)
  • Vanilla JavaScript (progressive enhancement)
  • ARIA roles, states, and properties where native HTML semantics are insufficient

The website has been tested with:

  • NVDA screen reader with Firefox (Windows)
  • VoiceOver with Safari (macOS and iOS)
  • TalkBack with Chrome (Android)
  • Keyboard-only navigation
  • Automated testing using axe-core

Formal Complaints Procedure

We are committed to addressing accessibility barriers promptly. If you experience an accessibility problem with this website, please:

  1. Contact us using the details below, describing the barrier you encountered and the URL of the page affected.
  2. We will acknowledge your report within 3 business days.
  3. We aim to resolve or provide a workaround for reported barriers within 30 days.

If you are not satisfied with our response, you may raise a complaint with the relevant national equality body or disability rights organisation.

Contact for Accessibility Issues

If you encounter any accessibility barriers on this website, or need content in an alternative format, please contact us:

Email: accessibility@eu4reconstruction.eu
Subject line: Accessibility Feedback — [page URL]

You may also use the general contact form, selecting "Accessibility & Website" as the enquiry type.

Improvement Roadmap

We regularly review and improve the accessibility of this website. Planned improvements include:

  • Full accessibility audit by an independent WCAG specialist (Q2 2025)
  • Remediation of all identified PDF documents (Q3 2025)
  • User testing with disabled users (Q3 2025)
  • Addition of a language selector for Ukrainian-language content (Q4 2025)
